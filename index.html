    QTodoTxt


    QTodoTxt is a cross-platform UI client for todo.txt files (see todo.txt).

    It allows you to manage projects, contexts in a GTD way.
    You can use it on Linux, Windows, Mac Os X and there are even IPhone and Android mobile App (right here) !

    Your todo.txt file can be sync with Dropbox, Owncloud or what you want, it's just a text file !

    Downloads

    Latest stable release (v1.4.0):

    • For Ubuntu/Debian: deb package
    • For Windows: Windows installer
    • For Mac OS X: Mac Os X installer
    • Everywhere: tar.gz archive

    For Debian/Ubuntu you can also add the following repo to your sources.list:

    echo "deb http://dl.bintray.com/mnantern/deb /" | sudo tee /etc/apt/sources.list.d/qtodotxt.list
