    QTodoTxt


    QTodoTxt is a cross-platform UI client for todo.txt files (see todo.txt).

    It allows you to manage projects, contexts in a GTD way.
    You can use it on Linux, Windows, Mac Os X and there are even IPhone and Android mobile App (right here) !

    Your todo.txt file can be sync with Dropbox, Owncloud or what you want, it's just a text file !

    News

    22 Mar. 2014: Version 1.3.0 released!

    I've just released v1.3.0 of QTodoTxt a cross-platform GUI for todo.txt.
    New features include better packaging (for Windows and Debian at least), a shortcut to change priority of selected tasks, a case insensitive search and a lot more.
    You can download this latest version on the project page.

    Feel free to try !
    I would also thank contributors to QTodoTxt: chaghi and dreuter.

    12 Jan. 2014: Version 1.2.0 released!

    I've just released v1.2.0 of QTodoTxt a cross-platform GUI for todo.txt.
    New features include support of FutureTasks, a Mac Os X package, a filter bar and a lot more.
    You can download this latest version on the project page.

    Feel free to try !
    I would also thank new contributors to QTodoTxt: chaghi, mgushee and CalumJEadie.

    13 Aug. 2013: Version 1.1.0 released!

    I've just released v1.1.0 of QTodoTxt.
    New features include an option to auto-archive done tasks, another one to enable auto-save of todo.txt and more.
    You can download this version on my GitHub page and some screenshots are available here.
    Feel free to try !

    19 Jun. 2013: Version 1.0.0 released!

    I've been searching for a cross-platform GUI for todo.txt for a while now but found nothing that match my criteria (utf8, due date, ...).
    So I've improved an initial work by David Elentok (thanks to him!) and I released v1.0.0 last week: download page
    Feel free to try !
